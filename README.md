# retail-student-survey-dashboard

Student Retail Spending Analysis Dashboard
Project Overview

This project presents a Power BI dashboard created using a Student Survey dataset from the retail industry.
The goal of the project is to analyze student purchasing behavior across different store locations, store settings, and product categories.

The dashboard provides insights into how students spend money on categories such as Video Games, Indoor Games, Outdoor Sports, Toys, Books, and Gadgets.

Tool Used

Microsoft Power BI

Dashboard Features
1. Tabular Visualization

Displays the Total Amount of Purchases (TAP) based on:

Store Location

Store Setting

Conditional formatting was applied:

Red → TAP < 35,000

Yellow → 35,000 ≤ TAP < 60,000

Blue → TAP ≥ 60,000

2. Matrix Visualization

Shows Outdoor Sports spending across different age groups and store settings (Rural, Suburb, Urban).

3. Funnel Chart

Displays Total Amount of Purchase by Store Setting with data labels shown as Percentage of First.

4. Pie Chart

Represents Total Amount of Purchase by Store Location for Suburban store settings only.

5. Scatter Plots

Two scatter plots were used to analyze spending patterns:

Video Games purchases vs Indoor Sports spending across ages

Video Games purchases vs Outdoor Sports spending across ages

Key Insights

Suburban stores generated the highest total purchases.

Spending on outdoor sports varies significantly across age groups.

Some age groups show higher interest in video games and sports-related products.

Dataset

Student Survey dataset containing:

Age

Store Location

Store Setting

Purchase amounts across product categories
